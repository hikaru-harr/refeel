# ReFeel

## 技術スタック

* **Server**: 
    - Hono

* **DB**: 
    - Prisma + PostgreSQL

* **Front**: 
    - React

* **Auth**:
    - Auth.js(jwt)

* **Test**:
    - Vitest

* **Lint/Format**
    - Biome
 
### 機能について

現状できていること
- ログイン
- 写真アップロード(単体)
  - 署名付きURLの取得
  - 取得した署名付きURLにてMinIOに直接アップロード
  - アップロード完了とともに完了通知
  - DB保存
  - 写真の取得
  - アップロードした写真一覧表示
  - 一覧写真クリックで詳細（拡大表示）
  - サーバー側でJWTトークンデコードして情報取得・利用
- 足りない部分
  - AIタグ付け
  - 検索
  - 各写真へのコメント
  - 各写真のお気に入り
  - EXIF/メタ情報抽出を追加
  - 一覧表示を年月別で表示
  - アルバム機能
  - 友人・家族共有

### 導入手順
- 環境変数の設定
- firebaseのシークレットキーの設定
- パッケージのインストール
- docker compose upにてDBとMinIOの立ち上げ
- マイグレート
